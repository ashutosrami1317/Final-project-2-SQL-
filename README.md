# University Database Management System

## 📖 Project Overview

This project demonstrates a **University Database Management System** built using SQL. It covers database creation, table relationships, data insertion, CRUD operations, filtering, joins, aggregate functions, subqueries, and reporting queries.

The project is designed for learning and practicing SQL database concepts.

---

## 📂 Database Name

`UniversityDB`

---

## 🗄️ Database Schema

The database contains the following tables:

- **Departments**
- **Students**
- **Courses**
- **Instructors**
- **Enrollments**

### Relationships

- A department offers multiple courses.
- A department has multiple instructors.
- Students enroll in courses through the Enrollments table.
- Courses belong to a department.

## 🚀 How to Run

1. Open MySQL Workbench or any SQL client.
2. Create a new SQL script.
3. Copy the contents of `Final Project.sql`.
4. Execute the script.
5. The database `UniversityDB` will be created automatically.
6. Run the included queries to test the database.
